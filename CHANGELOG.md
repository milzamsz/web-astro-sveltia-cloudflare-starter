## [1.0.0] — 2026-06-27

Initial public template release.

### Highlights

- Astro 7 + TypeScript, Tailwind CSS v4, monochrome OKLCH design tokens
- Bilingual content (English / Indonesian) with prefix-based i18n routing
- Sveltia CMS at `/admin` (GitHub backend, Cloudflare R2 media)
- Cloudflare Pages hosting with Pages Functions, D1, and R2
- Better Auth: email/password + mandatory TOTP 2FA, role-based admin access
- Contact form with Turnstile, D1 storage, and Resend email
- SEO: canonical, hreflang, JSON-LD, Open Graph, sitemap, RSS, dynamic `llms.txt`
- Blog, services, docs (Starlight), and marketing pages
- Lighthouse: 100 Performance / Accessibility / Best Practices / SEO on key pages
