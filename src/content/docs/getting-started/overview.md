A production-ready multilingual marketing website built with **Astro**, **Sveltia CMS**, and **Cloudflare Pages**.

## Features

- **Multilingual content** — Indonesian (default) + English with automatic locale switching
- **CMS-powered** — Sveltia CMS for editing pages, blog posts, services, and documentation
- **Performance** — Astro's zero-JS architecture with static generation and Cloudflare CDN
- **Security** — Turnstile CAPTCHA on contact forms, environment-scoped secrets, dependency review
- **SEO** — JSON-LD structured data, sitemaps, Open Graph meta, and a `robots.txt`

## Tech Stack

| Layer | Technology |
| --- | --- |
| Framework | [Astro](https://astro.build) v7 |
| CMS | [Sveltia CMS](https://sveltia.github.io/sveltia-cms/) |
| Styling | [Tailwind CSS](https://tailwindcss.com) v4 |
| Hosting | [Cloudflare Pages](https://pages.cloudflare.com) |
| Auth | [Cloudflare Access](https://www.cloudflare.com/teams/access/) |
